Linux でのビルド

無料プランをお使いの組織には毎月 400,000 クレジット (年間 2,880 ドル相当) が無料で付与され、リポジトリをパブリックにするだけで Linux コンピューティングでのオープンソース プロジェクトのビルドにご利用いただけます。

Linux でビルドを開始する

macOS でのビルド

無料プランでは、macOS でのオープンソース プロジェクトのビルド向けに、毎月 25,000 クレジット (年間 180 ドル相当) が付与されます。ご希望の方は billing@circleci.com までお問い合わせください。

CircleCI のブログ

CircleCI のブログでは、オープンソース プロジェクトでの CircleCI 活用方法について取り上げています。 ぜひこちらからお読みください。

オープンソース向け Orbs

Orbs からビルド済みのコマンド、ジョブ、Executors を設定ファイルにインポートすれば、
すべてをゼロから揃える必要はなくなり、時間の節約につながります。Orbs の一覧はこちら ▸

CircleCI CLI

circleci/circleci-cli

CircleCI コマンドライン インターフェイスをインストールして構成します。

Codecov

codecov/codecov

複雑な構成を行うことなく、カバレッジ レポートを Codecov にアップロードできます。

Node

circleci/node

Node.js プロジェクトをビルドおよびテストするための共通タスクが簡単に実施できるようになります。

Slack

circleci/slack

CircleCI のジョブの状況を知らせる Slack のカスタム通知を、簡単に作成し自動化することができます。

Heroku

circleci/heroku

Heroku CLI をインストールし、Heroku アプリを Git からデプロイします。

Cypress

cypress-io/cypress

CircleCI の構成に時間を取られることなく、Cypress.io のエンドツーエンドのブラウザー テストを実行できます。
