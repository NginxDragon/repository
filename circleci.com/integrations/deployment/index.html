Automate deployments with
CircleCI orbs

A fast, out-of-the-box solution for integrating with AWS Lambda, GCP, Kubernetes,
and other deployment services.

Start Building for Free Explore Orbs

Deployment solutions that are built for the cloud

  • Deployment agnostic

    Whether your team deploys to AWS, GCP, Azure, Heroku or other cutting edge technologies, CircleCI orbs help you automate and streamline deployment.

  • Clear version management

    Orbs update automatically to keep your pipeline running smoothly. You can also opt to pin an orb to a specific major, minor, or patch version for more manual control.

  • Increase organizational efficiency

    Orbs help you ensure consistency and easily roll out changes across your organization. With orbs, you can quickly share deployment patterns within your team, your company, or across your community.

Deployment to cloud services

Google Cloud CLI

circleci/gcp-cli

Install and configure the Google Cloud CLI (gcloud).

Kubernetes

circleci/kubernetes

A collection of tools for working with Kubernetes on CircleCI

Salesforce

circleci/salesforce-sfdx

Salesforce SFDX CLI integration for CircleCI. Easily create CI/CD pipelines for your Salesforce integrations.

Alcide.io

alcideio/alcide-advisor

Detect vulnerabilities and misconfiguration drifts early on in development when you run Kubernetes clusters.

Google Kubernetes Engine

circleci/gcp-gke

Deploy a Kubernetes cluster and update production code in seconds from your CI pipeline.

Azure Kubernetes Service

circleci/azure-aks

Ship faster, operate with ease, and scale confidently with Azure Kubernetes Service.

Red Hat OpenShift

circleci/redhat-openshift

Automate the build, deployment, and management of Kubernetes applications.

Kublr

kublr/kublr-api

Centrally deploy, run, and manage your large enterprise’s multi-cluster Kubernetes deployments across all of your environments.

Helm

circleci/helm

Find, share, and use software built for Kubernetes with Helm Charts.

DeployHub

deployhub/deployhub-orb

Automate the release steps of your workflow, creating a 100% repeatable deployment process for every state in your pipeline.

VMware Code Stream

vmware/codestream

Release higher quality applications and IT code faster while reducing operational risk

Pulumi

pulumi/pulumi

Enable a preview of any cloud infrastructure changes to be made to a Pulumi stack.

Convox

convox/orb

Deploy your application to the Convox platform as part of a CircleCI workflow.

Spinnaker

circleci/spinnaker

Simplify deployments with Spinnaker.

realMethods

realmethods/appgen

Generate MVP-quality apps that are instantly deployable to CircleCI.

LaunchDarkly

launchdarkly/ld-find-code-refs

Surface all references to feature flags in your code in your CI/CD process.

Rafay Systems

rafaysystems/rafay

Deploy a Kubernetes Cluster from CircleCI using the Rafay Systems Platform.

AWS CLI

circleci/aws-cli

Install and configure the AWS command-line interface (CLI). Easily interact with any AWS service from the command line.

AWS CodeDeploy

circleci/aws-code-deploy

Deploy applications to AWS CodeDeploy.

AWS ECS

circleci/aws-ecs

An orb for working with Amazon Elastic Container Service. Supports the EC2 and Fargate launch types and Blue/Green deployment via CodeDeploy.

Amazon EKS

circleci/aws-eks

Deploy, manage, and scale containerized applications using Kubernetes on AWS.

AWS SAM Serverless

circleci/aws-sam-serverless

Build, test, and deploy your AWS serverless applications on CircleCI utilizing the AWS Serverless Application Model.

Cloud Foundry

circleci/cloudfoundry

Push and deploy applications to Cloud Foundry.

Fairwinds

fairwinds/rok8s-scripts

Provides a framework for building GitOps workflows with Docker and Kubernetes.

Google Cloud Run

circleci/gcp-cloud-run

Build and deploy stateless images to Google Cloud Run as serverless applications.

Heroku

circleci/heroku

Install the Heroku CLI and deploy Heroku apps via Git.

Pantheon

pantheon-systems/pantheon

Push Drupal and WordPress code to Pantheon Dev and Multidev Environments.

Quali

quali/cloudshell-colony

Creates and terminates on-demand application environments for each stage of your pipeline.
